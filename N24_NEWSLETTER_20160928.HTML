Edito : le Web est là pour longtemps

À intervalles réguliers, des articles annoncent la mort prochaine du Web. Permettez-moi de douter. Les technologies Web n'ont jamais été aussi puissantes, elle n'ont jamais autant été utilisées. Alors oui, le site Web tel que nous le connaissons aujourd'hui va sans doute évoluer. Les applications Web vont sans doute se développer. Mais franchement, pensez-vous que des standards ouverts et appropriés comme HTML5, CSS et javascript vont disparaître ? Pensez-vous sérieusement que des technologies nouvelles pourront supplanter des mécanismes aussi puissants et éprouvés que le responsive design ? Pensez-vous sérieusement que tous les éditeurs de contenus Web pourront les diffuser via des applications ? Si tel est le cas, il y a fort à parier que celles-ci auront alors la couleur, le goût et un fonctionnement proche de celui de nos sites Web actuels. Le Web sera peut-être un peu moins visible qu'aujourd'hui, mais il sera là, et à mon avis, il est là pour longtemps. La transformation numérique, très à la mode en ce moment, consiste surtout pour les entreprises à améliorer l'expérience de leurs clients. Le Web et la culture Web vont jouer un rôle majeur pour atteindre cet objectif. Alors voilà, faites du Web et n'écoutez pas ceux qui parient sur sa disparition.

Élie Sloïm

Champs obligatoires et données recueillies

De manière intuitive, vous pourriez penser qu'en augmentant le nombre de champs obligatoires, vos formulaires vous permettront de recueillir une plus grande quantité de données. Cette étude de Microsoft montre qu'au contraire, réserver les mentions de champs obligatoires aux données qui le méritent vraiment augmente le nombre de réponses et conduit les utilisateurs à fournir plus d'informations dans les champs facultatifs.

Bonne pratique vue par l'expert

Ce mois-ci nous allons parler de la mention des champs obligatoires. La bonne pratique N°87 (v3) ou 39 (v2) indique que L'étiquette de chaque champ de formulaire indique si la saisie est obligatoire. Cette mesure semble tout à fait secondaire, et pourtant d'un point de vue accessibilité, elle peut s'avérer très importante pour les personnes qui mettent beaucoup de temps à saisir les formulaires. Au delà ce que cas spécifique, tout le monde est concerné, utilisateurs et administrateurs de sites. L'étude de Microsoft que nous avons citée en lien montre que si vous prenez soin de rendre obligatoire ce qui l'est vraiment, vous avez même de bonnes chances d'augmenter le taux de remplissage des champs facultatifs. C'est encore un très bel exemple de renforcement de la confiance par la qualité Web.
